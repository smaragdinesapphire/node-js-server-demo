# NodeJS Server Demo

## 中文

### 介紹

這是一份 NodeJS 起 server 的範例，包含:

- http request
- websocket

可以用[node-js-server-demo-client](https://github.com/smaragdinesapphire/node-js-server-demo-client)測試。

### 開發環境

- node@19.8.1
- nodemon@3.0.1

### 開始

```
// cmd
npm install
npm run dev
```
